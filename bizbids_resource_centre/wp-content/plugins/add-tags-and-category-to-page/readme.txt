This plugin allow you to use tags and category in pages and post type.

== Description ==

This Plugin allows user to use tags and category in wordpress pages and post types.

You can enable and disable tags and category support for pages under Settings > Add Tags And Category.

Note: We don't monitor support forum here. Please visit our <a href="http://dineshkarki.com.np/forums/forum/add-tags-and-category-to-page-and-post-types" target="_blank">support forum</a> for any issues. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the setting page of the plugin from Settings > Add Tags And Category For Page.
